# Platform Configuration Templates

このディレクトリには、アプリ公開用のプラットフォーム設定テンプレートが含まれています。

## 使用方法

1. `app_release_template.json` をコピー
2. ゲーム固有の設定に更新
3. 各プラットフォーム設定を自動生成

## 構成

- `app_release_template.json` - 汎用アプリ公開設定テンプレート
- `ios/` - iOS固有設定テンプレート
- `android/` - Android固有設定テンプレート  
- `docs/` - App Store用ドキュメントテンプレート# Platform Configuration Templates
